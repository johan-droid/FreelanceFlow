## Project Overview

FreelanceFlow is a landing page for a financial platform designed for freelancers. The original single-file HTML has been converted into a modern Next.js application with a component-based architecture.

## Key Components

- Navbar: Sticky navigation with scroll detection
- HeroSection: Dashboard mockup with interactive charts
- PainPointsSection: Problem cards with scroll animations
- FeaturesSection: Four interactive demos (calculator, tax estimator, timeline, chat)
- TestimonialsSection: Client testimonials with fade-in animations
- PricingSection: Pricing tiers with toggle and FAQ accordion
- SecuritySection: Trust and security cards with partner logos
- FinalCTASection: Call-to-action section
- Footer: Links and status information

## Running the Project

Development server: npm run dev
  → Open http://localhost:3000

Production build: npm run build
  → Start production server: npm start
